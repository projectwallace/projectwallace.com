---
title: Total rules
---

The number of rules in the CSS.

## Example

```css
/* rule 1 */
selector {
	property: value;
}

@media (all) {
	/* rule 2 */
	selector {
		property: value;
	}
}
```

There are **2 rules** in this example.

```js
{
	"rules.total": 2
}
```
